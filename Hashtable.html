Hash Tables

What is Hashing?

Hashing is a technique used to store and retrieve data efficiently A hash function converts a key into an index of an array 

Hash Function

h(key) = key % table_size

Example:

Key = 25
Table Size = 10

h(25) = 25 % 10 = 5
Store key 25 at index 5

Collision

A collision occurs when two different keys produce the same hash index 

25 % 10 = 5
35 % 10 = 5

Collision Occurs

Types of Hashing

  • Open Hashing (Separate Chaining)
  • Closed Hashing (Open Addressing)

Open Hashing (Separate Chaining)

In Open Hashing, each index stores a linked list When a collision occurs, the new element is inserted into the linked list

Open Hashing Operations

Insertion Algorithm

1. Compute hash index
2. Create new node
3. Insert node into linked list
4. End

Insertion Code

void insert(int key)
{
    int index = key % SIZE;

    struct Node *newNode =
    (struct Node*)malloc(sizeof(struct Node));

    newNode->data = key;
    newNode->next = table[index];

    table[index] = newNode;
}

Search Algorithm

1. Compute hash index
2. Traverse linked list
3. Compare each node
4. Return found/not found

Search Code

int search(int key)
{
    int index = key % SIZE;

    struct Node *temp = table[index];

    while(temp != NULL)
    {
        if(temp->data == key)
            return 1;

        temp = temp->next;
    }

    return 0;
}

Closed Hashing (Open Addressing)

In Closed Hashing, all elements are stored inside the hash table When a collision occurs, another free location is searched

Linear Probing Formula

h(k,i) = (h(k) + i) % size

Closed Hashing Operations

Insertion Algorithm

1. Compute hash index
2. If empty insert
3. Else probe next slot
4. Repeat until free slot found

Insertion Code

void insert(int key)
{
    int index = key % SIZE;

    while(hashTable[index] != -1)
    {
        index = (index + 1) % SIZE;
    }

    hashTable[index] = key;
}

Search Algorithm

1. Compute hash index
2. Compare current slot
3. Continue probing
4. Stop when found

Search Code

int search(int key)
{
    int index = key % SIZE;

    while(hashTable[index] != -1)
    {
        if(hashTable[index] == key)
            return 1;

        index = (index + 1) % SIZE;
    }

    return 0;
}
