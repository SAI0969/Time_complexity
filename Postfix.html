Postfix Expression

A postfix expression is an expression in which the operator comes after the operands

Infix Postfix
A+B AB+
A-B AB-
A*B AB*
(A+B)*C AB+C*
A+B*C ABC*+

Evaluation

  1. Create an empty stack
  2. Scan the expression from left to right
  3. If operator is found:
    • Pop two operands
    • Perform operation
    • Push result back into stack
  4. Continue untill expression ends
  5. Remaining element in the stack is the final answer

Algorithm

  1. Create an empty stack
  2. Scan the expression from left to right
  3. If symbol is an operand:
    • Push into stack
  4. If symbol is an operator:
    • Pop operand2
    • Pop operand1
    • result = operand1 operator operand2
    • Push result
  5. Repeat untill expression ends
  6. Pop final result from stack
  7. Display result

C Program

C Program
#include <stdio.h>
#include <ctype.h>

int stack[100];
int top = -1;

void push(int value)
{
    stack[++top] = value;
}

int pop()
{
    return stack[top--];
}

int main()
{
    char postfix[100];

    printf("Enter Postfix Expression: ");
    scanf("%s", postfix);

    int i = 0;

    while(postfix[i] != '\0')
    {
        char ch = postfix[i];

        if(isdigit(ch))
        {
            push(ch - '0');
        }

        else
        {
            int operand2 = pop();
            int operand1 = pop();

            switch(ch)
            {
                case '+':
                    push(operand1 + operand2);
                    break;

                case '-':
                    push(operand1 - operand2);
                    break;

                case '*':
                    push(operand1 * operand2);
                    break;

                case '/':
                    push(operand1 / operand2);
                    break;
            }
        }

        i++;
    }

    printf("Result = %d", pop());

    return 0;
}
