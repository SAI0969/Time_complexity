Binary Search Tree Operations

About Binary Search Tree

  1. A Binary Search Tree is a binary tree in which:
    • Left Subtree < Root < Right Subtree

    Example:

    
          50
         /  \
       30    70
      / \   / \
    20 40 60   80
    Operation Algorithm (Text / Code) Time Complexity
    Node Structure 
    struct Node
{
    int data;
    struct Node *left;
    struct Node *right;
};
    1. Create a node
    2. Store data in node
    3. Initialize left and right child pointers as NULL
    		No Time Complexity
    Create a new node

    Algorithm

    1. Allocate memory for node
    2. Store data
    3. Set Left = NULL
    4. Set Right = NULL
    5. Return Node

    Code

    struct Node* createNode(int data)
{
    struct Node *newNode =
        (struct Node*)malloc(sizeof(struct Node));

    newNode->data = data;
    newNode->left = NULL;
    newNode->right = NULL;

    return newNode;
}
    		 O(1) or O(n)
    Insertion

    Algorithm

    1. If tree is empty, Create a node
    2. If Data < Root, insert in the left subtree
    3. If Data > Root, insert in the right subtree
    4. Return root

    Code

    struct Node* insert(struct Node *root, int data)
{
    if(root == NULL)
        return createNode(data);

    if(data < root->data)
        root->left = insert(root->left, data);

    else if(data > root->data)
        root->right = insert(root->right, data);

    return root;
}
    		 O(log n)
    Search

    Algorithm

    1. Compare key with root
    2. If equal then print found
    3. If smaller then move left
    4. If greater then move right
    5. Repeat untill found or null

    Code

    struct Node* search(struct Node *root, int key)
{
    if(root == NULL || root->data == key)
        return root;

    if(key < root->data)
        return search(root->left, key);

    return search(root->right, key);
}
    		 O(log n)
    Inorder Traversal
    (Left -> Root-> Right)

    First visit Left Subtree, then Root, then Right Subtree

    Algorithm

    1. Traverse Left Subtree
    2. Visit Root
    3. Traverse Right Subtree

    Code

    void inorder(struct Node *root)
{
    if(root != NULL)
    {
        inorder(root->left);
        printf("%d ", root->data);
        inorder(root->right);
    }
}
    		 O(n)
    Preorder Traversal
    (Root -> Left-> Right)

    First visit Root, then Left Subtree, then Right Subtree

    Algorithm

    1. First visit Root
    2. Traverse Left Subtree
    3. Traverse Right Subtree

    Code

    void preorder(struct Node *root)
{
    if(root != NULL)
    {
        printf("%d ", root->data);
        preorder(root->left);
        preorder(root->right);
    }
}
    		O(n)
    Postorder Traversal
    (Left -> Right -> Root)

    First visit Left Subtree, then Right Subtree, then Root

    Algorithm

    1. Traverse Left Subtree
    2. Traverse Right Subtree
    3. Visit Root

    Code

    void postorder(struct Node *root)
{
    if(root != NULL)
    {
        postorder(root->left);
        postorder(root->right);
        printf("%d ", root->data);
    }
}
    		O(n)
    Deletion of a node

    Algorithm

    1. If you want to delete Leaf Node, Delete directly
    2. If you want to delete One Child,Replace node with child
    3. If you want to delete Two Children, Replace with
    inorder successor and Delete successor

    Code

    struct Node* deleteNode(struct Node *root, int key)
{
    if(root == NULL)
        return root;

    if(key < root->data)
        root->left = deleteNode(root->left, key);

    else if(key > root->data)
        root->right = deleteNode(root->right, key);

    else
    {
        if(root->left == NULL)
        {
            struct Node *temp = root->right;
            free(root);
            return temp;
        }

        else if(root->right == NULL)
        {
            struct Node *temp = root->left;
            free(root);
            return temp;
        }

        struct Node *temp =
            findMin(root->right);

        root->data = temp->data;

        root->right =
            deleteNode(root->right,
                       temp->data);
    }

    return root;
}
    		O(log n)
